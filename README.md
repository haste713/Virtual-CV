A modern, responsive, and interactive personal portfolio website showcasing my academic background, technical skills, projects, certifications, and professional experience as an Information Technology student.

This portfolio serves as my digital CV, allowing recruiters, employers, and collaborators to explore my work, learn more about me, and access my GitHub repositories and LinkedIn profile.
